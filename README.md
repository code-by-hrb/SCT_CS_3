🛡️ SCT_CS_3: ShieldGuard
Ultra-secure, client-side credential analyzer & local vault. Evaluate password resilience using mathematical entropy formulas and secure your generated keys entirely in your browser.

💼 SkillCraft Technology Internship Project | Cyber Security Task #3

🎯 Main Goal
To provide a 100% private, serverless hub for checking, generating, and logging secure passkeys locally. By calculating cryptographic strength metrics entirely within the browser client, ShieldGuard ensures your master credentials never touch a network or risk server-side exposure.

🔥 Core Features
✅ Live Strength Analysis: Real-time validation of password complexity using visual meter tracking against strict multi-rule security criteria.

✅ Cryptographic Metrics: Instant mathematical calculations for total Shannon Entropy (bits) and estimated Time-to-Crack speeds.

✅ Automated Secure Generator: One-click production of high-entropy, randomized alphanumeric and special character strings.

✅ Isolated Encrypted Vault: A local historical session log that automatically queues unique generated or verified records without duplicating identical entries.

✅ Responsive Customizer UI: A sleek, modern dual-theme workspace featuring seamless dark/light mode toggles and full layout responsiveness.


🛠️ Tech Stack
- HTML5: Structuring the core application views, dynamic tables, and responsive navigation components.
- CSS3: Implementing flexible layout grids, custom properties for instant dark/light themes, and interactive status indicators.
- JavaScript (ES6): Driving the live cryptographic analysis engine, clipboard mutations, state deduplication, and single-page view routing.
  

🚀 Quick Start
1. Clone this repository to your local workspace.
2. Open `index.html` directly in any modern web browser.
3. Input or generate a key, evaluate its defensive integrity, and safely manage your session vault history!
